# TMF Target Audience

## Primary Persona: Non-Technical AI Enthusiast

**Who they are:**
- Heard about OpenClaw from a technical friend, YouTube, or Twitter/X
- Excited about having a personal AI assistant
- Cannot install or configure it themselves (CLI, Docker, API keys are foreign)
- Willing to pay for someone to set it up
- Individuals, not businesses

**The adoption funnel (where they get stuck):**
```
Hears about OpenClaw .............. 100%
Watches a YouTube tutorial ........ ~60%
Attempts installation ............. ~20%
Gets it running ................... ~10%
Uses it productively .............. ~3-5%
Uses it daily ..................... ~1-2%
```

TMF's three tiers address every drop-off point in this funnel.

## Pain Points

### Installation Barriers
- Requires Node.js, Docker, terminal/CLI knowledge
- API key management across multiple providers
- Docker containers crash with no user-friendly error messages
- No visual feedback during processing
- VPS provisioning and deployment is a foreign concept

### Post-Installation Barriers
- Managing the system is harder than installing it
- No idea what to DO with the AI once it's running
- Security mistakes (weak credentials, exposed instances)
- Channel connections are complex (WhatsApp QR codes, Telegram bot tokens)
- No structured workflow — just a chatbot with no system behind it

### The Real Problem
They don't just need installation — they need a **system**. A chatbot that answers questions is useless for productivity. A chatbot connected to a task board with scheduled rituals, operational modes, and structured workflows is transformative.

## Secondary Persona: Technical Person Who Lacks the Methodology

- Can install OpenClaw themselves
- Has it running but uses it as a basic chatbot
- No structured time management workflow
- Would buy Package 1 ($50 DIY) for the methodology and templates alone

## Demographic Signals
- Age: 25-55
- Income: middle to upper-middle class (can afford $50-$400 one-time)
- Tech comfort: uses apps daily but doesn't code
- Motivation: overwhelmed by tasks, wants to feel in control
- Discovery: word of mouth from technical friends, YouTube, LinkedIn, Twitter/X
