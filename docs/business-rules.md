# TMF Business Rules

## Pricing Model
- **One-time purchase only** — no subscriptions, no recurring fees
- This is a key differentiator; all SaaS competitors charge $8-99/month
- Three tiers: $50 / $150 / $400

## Target Market
- B2C only — individuals, not businesses
- Non-technical people who want AI assistants but can't set them up
- The "heard about it from a friend" demographic

## Product Boundaries
- TMF configures existing tools (OpenClaw + Trello) — it does NOT build new tools
- TMF includes methodology (modes, rituals, board structure) — not just technical setup
- Security hardening is standard in the Done-For-You tier
- The $400 tier includes a review of existing OpenClaw problems/bugs

## Technical Defaults
- Default task board: user's main board (Vitaly's is "2026")
- New tasks go to "Inbox/Brain Dump" unless specified
- Due dates use user's local timezone offset, NEVER UTC (Z)
- Card titles: short and actionable; details go in description
- Brain dumps are saved for periodic pattern scanning, NOT treated as immediate to-dos

## Content Rules
- Every client interaction can become content (with permission)
- Index all tasks and memories in English, even if discussed in another language
- LinkedIn is the primary content channel; YouTube and Skool are secondary

## Development Philosophy
- **Lowest effort first. Ship. Iterate.** — build as an organism that grows
- Start with the simplest version, then improve, then improve again
- Avoid perfectionism — it causes freeze and missed windows
- If it's not needed on Day 1, it ships later
- Hormozi framework: lead magnet (free PDF checklist) → proper offer → packages

## MVP Scope Decisions
- **Dashboard/visualization**: Ship later. Trello's built-in reporting is enough for now. Dashboard only has value after months of data — useless on Day 1
- **AI coaching tone**: Two SOUL.md variants (tough love / gentle guide). One setup question picks the variant. Minimal effort
- **Calendar integration**: Skip for MVP. User tells AI their availability verbally. Zero integration work
- **ClickUp support**: Later. Trello only for now
- **Lead magnet**: Free PDF checklist — "Is Your AI Actually Helping You?" 10-point checklist targeting OpenClaw users/aspirants. Laser-focused on the niche. A/B test titles later
- **Landing page**: Gumroad product page for MVP (lowest effort). No custom site needed yet
- **Future website**: planning.art — brand home for TMF and future products. Client reviews, tips & tricks, planning techniques library. Long-term vision, not MVP

## Packages (Revised)
- **Package 1 ($50)**: DIY digital download via Gumroad. No support. Semi-technical users figure it out
- **Package 2 (TBD pricing)**: Skool community subscription (Agent Orchestrator). Covers TMF help + general OpenClaw/AI help. Details TBD — ships after Package 1 and 3
- **Package 3 ($400)**: VIP 90-min 1:1 session via Gumroad. Email support for now

## Onboarding
- OpenClaw asks a series of questions on first conversation: life areas, biggest time challenge, coaching style preference (tough love / gentle)
- Saves answers to memory, configures itself accordingly

## Trello Defaults
- 3 default backlogs: **Work**, **Personal/Home**, **Side Project**
- User can add/rename unlimited backlogs — OpenClaw adapts automatically
- Trello skill: cleaned-up version of Vitaly's working skill file, personal data removed, comments added

## Planning Session Guidelines
- **Daily**: ~15 min recommended
- **Weekly** (Sunday night): ~30 min recommended
- **Monthly** (last day of month): ~90 min recommended
- These are recommendations, not hard limits — users can go over
- OpenClaw reminds when approaching the time limit and does periodic time checks
- Acts like a good PM: keeps things on track, prevents side-tracking, but doesn't force stops
