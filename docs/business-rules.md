# TMF Business Rules

## Pricing Model
- **One-time purchase only** — no subscriptions, no recurring fees
- This is a key differentiator; all SaaS competitors charge $8-99/month
- Three tiers: $50 / $150 / $400

## Target Market
- B2C only — individuals, not businesses
- Non-technical people who want AI assistants but can't set them up
- The "heard about it from a friend" demographic

## Product Boundaries
- TMF configures existing tools (OpenClaw + Trello) — it does NOT build new tools
- TMF includes methodology (modes, rituals, board structure) — not just technical setup
- Security hardening is standard in the Done-For-You tier
- The $400 tier includes a review of existing OpenClaw problems/bugs

## Technical Defaults
- Default task board: user's main board (Vitaly's is "2026")
- New tasks go to "Inbox/Brain Dump" unless specified
- Due dates use user's local timezone offset, NEVER UTC (Z)
- Card titles: short and actionable; details go in description
- Brain dumps are saved for periodic pattern scanning, NOT treated as immediate to-dos

## Content Rules
- Every client interaction can become content (with permission)
- Index all tasks and memories in English, even if discussed in another language
- LinkedIn is the primary content channel; YouTube and Skool are secondary

## Market Timing
- Window is NOW (Feb 2026) — OpenClaw hype is at peak
- Gap between demand and capability is at maximum
- Gap will shrink as: OpenClaw improves onboarding, managed hosting matures, OpenAI potentially builds consumer version
- Speed to market matters more than perfection
